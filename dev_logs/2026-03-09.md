# Dev Log — 2026-03-09

## 前端战斗轮测试反馈

### Bug #1: 缺少交战选项 + 无法攻击未交战敌人
- **现象**: 闪避敌人后，敌人在同一地点但未交战，无法再对其发动攻击
- **原因**: 前端 Fight 只查找 `threat_area`（已交战敌人），忽略了地点上的未交战敌人
- **规则**: 调查员可以攻击同一地点的任何敌人（未交战的需先自动交战）
- **修复**:
  - 前端增加 Engage 按钮
  - Fight 目标选择展示所有地点敌人（交战+未交战）
  - 后端 FIGHT 行动对未交战敌人自动先交战

### Bug #2: 技能卡无法投入检定
- **现象**: 技能检定直接结算，没有投入手牌的交互步骤
- **原因**: 前端没有实现 ST.2（投入卡牌）步骤
- **规则**: 在技能检定中，可以从手牌投入技能卡（以及带匹配图标的其他卡），图标加到检定值上
- **修复**:
  - 前端增加「投入卡牌」弹窗（战斗/调查/闪避时触发）
  - 展示手牌中可投入的卡牌及其技能图标
  - 将选中的卡牌 ID 随行动一起发送给后端
